# JS 台灣雞蛋庫存雜貨門市分佈圖

國內蛋價又再飆新高，加上各地出現缺蛋潮，如今想吃到蛋可能得花點時間去各雜貨門市尋找。

## 使用方式
- 把整個專案複製一份到你的電腦裡，這裡指的「內容」不是只有檔案，而是指所有整個專案的歷史紀錄、分支、標籤等內容都會複製一份下來。
```sh
$ git clone
```
